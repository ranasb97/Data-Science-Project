# Vehicle Performance & Fuel Efficiency Analysis

## Overview
This project analyzes automobile performance and fuel efficiency data using Python and SQL. The goal of the project is to identify relationships between vehicle characteristics such as horsepower, weight, displacement, and fuel economy (MPG).

The project includes:
- Data cleaning and preprocessing
- Exploratory data analysis
- Data visualization
- SQL-based queries for insight generation

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- SQL

---

## Dataset Features
The dataset contains information about vehicles including:
- MPG (Miles Per Gallon)
- Horsepower
- Weight
- Engine displacement
- Cylinders
- Model year
- Origin

---

## Project Features

### Data Cleaning
- Removed missing values
- Converted numerical columns into proper numeric formats
- Cleaned dataset for analysis

### Data Analysis
- Calculated average MPG
- Identified highest horsepower vehicles
- Compared MPG across cylinder groups

### Data Visualization
Created visualizations including:
- MPG distribution histogram
- Horsepower vs MPG scatter plot
- Weight vs MPG scatter plot

### SQL Analysis
Used SQL queries to:
- Find average MPG
- Identify highest horsepower vehicles
- Compare MPG by vehicle origin
- Analyze heavier vehicle trends

---

## Key Insights
- Vehicles with higher horsepower generally have lower MPG
- Heavier vehicles tend to be less fuel efficient
- Smaller engines and lower cylinder counts often achieve better fuel economy
